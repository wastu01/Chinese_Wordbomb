# Chinese Wordbomb

中文詞語接龍遊戲

## Project Layout

- `dict_moedict_clean.json` — Reference dictionaries derived from the [萌典](https://github.com/g0v/moedict-webkit) open data dump.

- `dict_moedict_clean_s.json` simplified chinese version.
